IT Help Desk - Amazon Connect/ Lex

More coming soon...

In this project, I created an IT help desk service using Amazon Connect and Amazon Lex. This service is able to offer customers a chat and voice experience. I first set up an Amazon Connect instance, and tested a sample voice and chat experience. Next, I set up an Amazon Lex bot that could find out the reason of each customer's call. I then customized a flow to use the customer's input to route them to the right queue on Amazon Connect. I was able to create an IT Help Desk contact center that uses the features in Amazon Lex....
